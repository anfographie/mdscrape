# instrumentationHook

The experimental `instrumentationHook` option allows you to set up instrumentation via the [`instrumentation` file](/docs/14/app/api-reference/file-conventions/instrumentation) in your Next.js App.

next.config.js

```
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
